## Browser object

1. window
```window``` is a global varible

property: 
* ```innerWidth```
* ```innerheight```
* ```outerWidth```
* ```outerHeight```

2. navigator

* ```navigator.appName```：browser name；
* ```navigator.appVersion```：browser version；
* ```navigator.language```：language；
* ```navigator.platform```：system type；
* ```navigator.userAgent```：User-Agent string
